
# Calculadora Simples em React Native

Aplicativo de uma simples calculadora, na qual realiza operações básicas.




## Funcionalidades

- Temas dark e light
- Operações:
    - Soma;
    - Subração;
    - Multiplicação;
    - Divisão.


## Stack utilizada

**React-Native**\
**Typescript**\
**Expo**
